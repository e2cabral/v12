# Glossário

## Feature

Unidade principal de organização do `v12`.

## Module

Agregador de providers, routes e extensão de uma feature.

## Service

Camada de regra de negócio.

## Repository

Camada de abstração de persistência.

## RequestContext

Objeto com `request`, `reply` e `container`.

## Plugin

Unidade de extensão do app.
