# Contribuição

Resumo curto

Esta página descreve como contribuir com o `v12` sem perder consistência arquitetural.

## Setup

```bash
npm install
npm test
npm run typecheck
```

## Padrões

- preserve feature-first
- mantenha APIs pequenas e explícitas
- adicione documentação junto com mudança estrutural

## Pull Requests

- explique o problema
- documente trade-offs
- inclua teste quando mudar comportamento
