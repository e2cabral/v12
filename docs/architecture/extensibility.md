# Extensibilidade

Resumo curto

O `v12` foi pensado para crescer por plugins, adapters e convenções.

## Pontos de extensão

- plugins
- providers
- adapters de banco
- guards
- event listeners
- jobs

## Boas práticas

- prefira extensão por plugin antes de modificar o core
- mantenha contracts estáveis
