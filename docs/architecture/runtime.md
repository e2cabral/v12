# Runtime

Resumo curto

Runtime descreve os componentes vivos do `v12` durante a execução.

## Componentes

- Fastify instance
- Container
- EventBus
- Module registry
- Route registry

## Performance

O runtime busca custo baixo por request e composição explícita.
