# Conceitos Fundamentais

Resumo curto

Esta seção explica os conceitos-base que estruturam o runtime e o modelo mental do `v12`.

## Quando usar

Use esta seção no onboarding ou quando precisar entender o comportamento interno antes de expandir uma aplicação.

## Conceito

Os conceitos centrais do `v12` são:

- containers
- modules
- services
- context
- lifecycle
- dependency graph
- execution
- configuration

## Exemplo rápido

```txt
Request
  -> Module
  -> Route
  -> Controller
  -> Service
  -> Repository
```

## Explicação completa

O `v12` tenta reduzir “mágica invisível”. Cada conceito possui fronteira clara e documentação própria.
