# i18n API

O V12 possui suporte nativo a internacionalização (i18n), permitindo gerenciar traduções globais e locais por módulo.

## Configuração Global

Habilite o i18n no `createApp` passando as opções desejadas.

```ts
const app = await createApp({
  i18n: {
    defaultLocale: 'pt-BR',
    localesPath: './locales', // Caminho opcional para arquivos JSON de tradução
  }
});
```

## Tradução por Módulo

Você pode definir traduções específicas dentro de um módulo. Estas traduções serão mescladas com as globais.

```ts
export const BillingModule = defineModule({
  name: 'billing',
  i18n: {
    'pt-BR': {
      invoice: 'Fatura',
      paid: 'Pago'
    },
    'en-US': {
      invoice: 'Invoice',
      paid: 'Paid'
    }
  }
});
```

## Uso no Contexto de Requisição

Dentro de handlers e middlewares, você tem acesso ao helper `t` através do `RequestContext`.

```ts
router.get('/invoice/:id', {
  handler: async ({ t, request }) => {
    return {
      message: t('billing.invoice') // Retorna 'Fatura' se o locale for pt-BR
    };
  }
});
```

O V12 tenta detectar o locale automaticamente através:
1. Header `Accept-Language`
2. Query parameter `locale` ou `lang`
3. Cookie `locale`
4. Locale padrão definido na configuração

## I18nService

Você também pode injetar o `I18nService` em qualquer classe.

```ts
import { I18nService } from 'v12';

export class MyService {
  constructor(private i18n: I18nService) {}

  doSomething() {
    const text = this.i18n.translate('messages.welcome', 'pt-BR', { name: 'João' });
  }
}
```

### Métodos do I18nService

- `translate(key, locale, args)`: Traduz uma chave para o locale especificado, suportando interpolação de argumentos.
- `addTranslations(locale, data)`: Adiciona dinamicamente novas traduções.
- `loadTranslationsFromPath(path)`: Carrega arquivos `.json` de um diretório.

## Interpolação

O V12 suporta interpolação simples usando `{{}}`.

**JSON:**
```json
{
  "welcome": "Bem-vindo, {{name}}!"
}
```

**Código:**
```ts
t('welcome', { name: 'Junie' }); // "Bem-vindo, Junie!"
```
